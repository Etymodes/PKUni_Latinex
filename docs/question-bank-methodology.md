# 教材完整题库：编制与质量说明

## 1. “完整”的范围

本题库以章节知识域完整覆盖为目标，而不是把教材逐页或逐题复制进网站。

- Wheelock's Latin：40 章全部进入覆盖矩阵；按章节记录形态、句法、词汇和阅读重点。
- Lingua Latina per se Illustrata: Familia Romana：35 章全部进入覆盖矩阵；利用书末 `INDEX VOCABVLORVM` 和 `INDEX GRAMMATICVS` 复核词汇首次出现及语法分布。
- 北京大学公开考试要求：作为难度域和组卷边界；教材章节顺序只表示教学阶段，不直接等于考试等级。

教材只用于确定学习内容、词头、结构和先后次序。网站中的题干、例句、模拟篇章、解析与参考译文均重新编写；不复制教材练习，不将模拟材料标成真题。

## 2. 难度域

| 难度 | 用途 | 主要内容 |
| --- | --- | --- |
| 初级 | 对应初级考试完整语法范围 | 基础词汇、变格变位、分词与不定式、核心虚拟式从句、Caesar/Nepos 式散文英译 |
| 中级 | 对应中级考试特殊形态与复杂句法 | 变体词形、目的分词、间接话语、主句虚拟式、主观理由、Cicero/Sallust 式长句 |
| 混合难度 | 跨域诊断和复习 | 从初级、中级与进阶题中随机抽取 |
| 进阶 | 超出最低考纲的原典能力 | 古式、语用、篇章信息层级、格律术语和复杂周期句 |

重要说明：Wheelock 后半册仍属于北大初级考试所要求的完整核心句法，因此以“核心/全范围”标记教学阶段，而不会机械地标成中级。LLPSI XXXIV–XXXV 的诗体和语法术语另归进阶。

## 3. 题型

- 词汇释义：从词典形识别语义。
- 词典回溯：从屈折形式返回词头，训练无辅助阅读时的查词能力。
- 形态识别：数、格、性、人称、时态、语气、语态和特殊词形。
- 句法与句式：从句功能、非限定形式、虚拟式语用、关联结构和长句骨架。
- 古典语体：Caesar、Nepos、Cicero、Sallust 常见叙述或演说特征。
- 翻译自评：先划分主干与从句，再对照英文参考译文和中文辅助说明。
- 正式制式模拟：四套约 180 词原创篇章，初级与中级各两套；180 分钟组卷只抽取一篇完整英译任务。
- 知识诊断组卷：客观题与短翻译混合，适合定位薄弱知识域。

## 4. 来源标签

每题至少保留一种来源标签：教材章节、北大考试范围，或仿写的古典作者语体。`仿 Caesar 原创`、`仿 Sallust 原创` 等标签表示依据作者常见结构重新创作，绝不表示该段出自原作或历年试卷。

## 5. 自动质量门槛

构建网站时会执行题库校验。以下问题会直接导致构建失败：

- 题目 ID 重复；
- 题干、解析或来源为空；
- 选择题不是四个互不相同的选项；
- 正确答案索引越界；
- 翻译自评题缺少拉丁语原文或参考译文。

内容层面的复核还包括：词典形是否规范、屈折形式能否回溯、答案位置轮换、正式模拟篇幅，以及难度标签是否符合考纲而非教材页码。

## 6. 继续扩充

管理员后台可新增、停用或覆盖题目。新增题建议沿用 `教材/章节 + 知识点 + 考试域 + 原创/原典` 的来源格式，并把完整原典的书名和章节号写入来源字段。若未来取得可核验且允许公开的历年试卷，应另设“真题”来源类别，和当前模拟题严格分开。
